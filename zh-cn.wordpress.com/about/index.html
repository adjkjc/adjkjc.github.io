关于我们

加州圣克鲁兹的 Automattician

在 WordPress.com，我们的使命是一次发布一个网站，以实现民主化。

开源 WordPress 是 最流行的在线发布平台，目前为超过 20% 的 Web 提供支持。 我们希望将 WordPress 体验带给更广泛的受众，因此我们在 2005 年创建了 WordPress.com。

我们是一款托管版本的开源软件。在这里，您无需具备任何技术知识即可开始撰写博客或在数秒内构建网站。

Automattic 的所有人都致力于不断改进 WordPress.com。我们几乎每天都推出更新并开发其他服务（如 Gravatar 和 Simplenote）以增强您在 Web 上的体验。我们希望构建您喜欢使用的产品、功能和主题，所以您随时都可以向我们提供您的反馈意见。

WordPress.com 上的几乎所有内容都是免费的，当前免费提供的内容在将来仍会免费提供。我们通过为定制设计和定制域等内容提供升级以及提供反垃圾邮件软件 Akismet 等产品和与大型媒体机构的 VIP 托管合作伙伴关系来使您的站点保持可用。

无论您是博客用户还是网站所有者，我们知道您可以在许多地方推介您的在线站点。如果您是当前用户，感谢您选择我们 — 我们喜欢和您在一起。如果您打算构建您的在线站点，但是还没找到好地方，不妨试试我们的服务。我们希望成为您在 Web 上的栖息地。
