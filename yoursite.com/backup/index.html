    Cloud Backup Frequently Asked Questions

    What does cloud backup mean?

    A cloud backup or cloud computer backup is the process of copying files, applications, virtual machines, or servers, which are then stored or backed up virtually. This backed up information is accessible using the internet.

    “The cloud” is a term referring to a network and data centers available online.

    Is 1TB backup the maximum amount of storage I can order?

    No, it is possible to purchase more than 1TB backup. Since your need for both storage and backup may be different, Yoursite.com makes it possible to choose the amount of backup you require. Our cloud backup plans can facilitate 50 GB, 100 GB, 250 GB up to 100 TB.

    Is Internet Explorer 9 browser supported on this platform?

    No. However, we support the following browsers: Internet Explorer 10, Microsoft Edge 25, Opera 16, Mozilla Firefox 23, Google Chrome 29, Safari 5.1.7, Safari 8 on macOS and IOS, or later versions of these browsers.

    Is it possible to back up databases such as Microsoft SQL Server?

    Yes, it is possible to back up databases such as Microsoft SQL Server. Since, Acronis cloud backup uses Microsoft Volume Shadow Copy Service, it can back up Microsoft Active Directory, Microsoft Exchange, Microsoft SQL Server, and Microsoft SharePoint in a consistent state.

    Acronis Backup Cloud also integrates special backup agents to facilitate Microsoft Exchange and Microsoft SQL Server to initiate instant restore service.

    Can I restrict the network usage for backups?

    Yes. You can allocate specific amounts of bandwidth for each backup plan.

    Are file-level recoveries from the (image) backups in the local storage possible?

    Yes, it is possible to do so.
