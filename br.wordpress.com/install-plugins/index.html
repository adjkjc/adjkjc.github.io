Traga o poder do Jetpack para o seu serviço de hospedagem favorito

Monitoramento de desempenho, compartilhamento social, SEO e outros recursos incluídos no WordPress.com estão disponíveis para todos. Se você estiver usando o WordPress com outro serviço de hospedagem, pode obter os mesmos recursos instalando o Jetpack.

Jetpack é a caixa de ferramentas completa do WordPress para design, marketing e segurança descomplicados. Seja para construir o seu site com o WordPress.com ou para criá-lo em seu próprio servidor, o Jetpack oferece as ferramentas necessárias para você ter sucesso em um só lugar.
