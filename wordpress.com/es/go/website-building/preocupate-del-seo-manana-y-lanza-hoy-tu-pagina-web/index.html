Preocúpate del SEO mañana y lanza hoy tu página web

19 mayo 2020
Accsky

En este artículo

  • Te explicamos qué es el SEO
  • Define tu misión
  • Convierte los clics en ventas

Una de las mejores maneras (y de las más fáciles) de promocionar tu negocio es creando una página web. Según la Oficina de los EE. UU. para la pequeña empresa, en Estados Unidos hay 28 millones de pequeñas empresas, por lo que es esencial que hagan todo lo que esté en sus manos para atraer clientes. Aunque haya quien diga que no se puede hacer una página web sin tener conocimientos sólidos de SEO (search engine optimization, posicionamiento en motores de búsqueda) para páginas web, lo más importante para el empresario de un pequeño negocio es conseguir tener una presencia en internet significativa.

¿Conclusión? Para cualquier persona que quiera tener un negocio viable hoy en día tener una página web no es negociable. Por eso hemos recopilado una selección de ideas de expertos sobre cómo crear una página web y por qué no es necesario contar con el SEO para poder lanzarla.

Te explicamos qué es el SEO

Entender bien cómo funciona el SEO permite que tu página web aparezca en la primera página de los resultados de búsqueda de Google cuando alguien escribe una palabra clave concreta (por ejemplo, si tienes una tienda de productos ecológicos querrás aparecer en los resultados de búsqueda cada vez que alguien de tu zona busque “alimentos ecológicos”). Los motores de búsqueda como Google utilizan algoritmos informáticos para decidir qué ven los usuarios cuando buscan algo. Nadie sabe con exactitud qué tiene en cuenta el algoritmo de Google, pero cada vez que se actualiza cunde el pánico entre los propietarios de páginas web, según Search Engine Land.

¿Por qué? Porque Google tiene 1.170 millones de usuarios, según Statista, y con que sólo una pequeña parte de esos usuarios haga clic en tu página web tu negocio puede crecer mucho. Esto lo corroboran también diferentes estudios: el 28 % de las búsquedas en internet de algo cercano terminan en compra, según la compañía de marketing HubSpot.

Un buen SEO implica tener contenido de calidad con palabras clave relevantes y ciertas consideraciones sobre el servidor de la web, como la rapidez con la que se cargan las páginas, si la página web funciona bien y con qué facilidad se navega. Por suerte, en WordPress.com nuestros temas ya incorporan todas estas funciones, así que no tienes que preocuparte por la velocidad de carga ni el tiempo de actividad. Pero recuerda: el SEO para páginas web sigue siendo una ciencia imperfecta, así que ¿por qué esperarte a tener conocimientos enciclopédicos sobre SEO? Empieza ya y lanza tu página web hoy mismo.

Define tu misión

Antes de desarrollar una página web pregúntate para qué la estás creando. ¿Es sólo para promocionar tus servicios o para crear un blog y atraer a más clientes? ¿Vas a usar la página web para generar ventas y dar a tus clientes la opción de comprar por internet en lugar de ir a la tienda? Cuando sepas tus objetivos generales sabrás cómo estructurar tu página web, pero eso no es lo único que tienes que tener en cuenta. También tendrás que:

  • Elegir un nombre. En WordPress.com encontrarás todo lo que necesitas para lanzar una página web. Necesitarás un nombre de dominio, o la URL de la página web, que es lo que hay que teclear para acceder a la web y descubrir tu negocio. Tu URL puede ser algo tan directo como tu nombre o el nombre de tu empresa —santiagolopez.com o magdalenasdesalamanca.com, por ejemplo— pero sólo si tienes un nombre único. Si no, en cualquier caso, tiene que ser una URL corta, con palabras clave y fácil de recordar. Todas las páginas web de WordPress.com incluyen una dirección WordPress.com gratuita en la URL, pero puedes conseguir una URL única si adquieres un plan.
  • Decidir qué secciones va a tener tu página web. Si vas a crear una página web para una tienda es posible que quieras incluir una página de contacto, una página con horarios de apertura y la dirección, una página de productos y servicios, así como una página con información sobre la empresa. Algunos empresarios también deciden crear un blog para mostrar sus conocimientos y conectar con sus seguidores. Aprovecha las herramientas del blog y las plantillas de diseño que puedes incorporar fácilmente a tu página web para no tener que empezar de cero.
  • Pensar en el aspecto visual. Para que una página web sea atractiva necesita elementos visuales. Cuelga fotos y vídeos de tu empresa y de los productos que más se venden, o entrevistas a cámara en las que expliques el origen y la historia. Puedes grabar vídeos con tu teléfono o buscar imágenes gratuitas en páginas web con fotos de archivo, pero lo mejor es hacer fotos propias para dar una sensación de autenticidad.
  • No usar mucho texto. Una página web con mucho texto puede desanimar, así que intenta ser conciso. Usa un lenguaje accesible y céntrate en lo que hace que tu empresa sea diferente, para ello resulta eficaz usar testimonios y un buen portafolio. Utiliza temas de empresa y portafolio gratuitos, y previsualiza cuál encaja mejor con tu página web. Si por ejemplo tienes una empresa de planificación de bodas, quizá la mejor opción para destacar lo que ofreces sea un tema que incluya un control deslizante de pantalla completa para poder para mostrar fotos en gran tamaño de los servicios que ya has realizado.

Convierte los clics en ventas

A fin de cuentas, una página web es un generador de clientes potenciales para el pequeño negocio. Puede atraer tráfico a tu tienda o convertir clics en ventas. De alguna manera, una página web ofrece a los usuarios la oportunidad de visitar tu empresa y de saber qué haces, así como los servicios exclusivos que ofreces antes de que tengan que invertir tiempo en visitar la tienda en persona. Las páginas web permiten “probar antes de comprar”, así que es importante para el pequeño negocio ofrecer online su mejor presencia en formato digital.

Por lo tanto, primero pon en marcha tu página web para mostrar el valor añadido que ofrece tu empresa a tus seguidores potenciales y así tendrás tiempo suficiente para pensar en el SEO después.
