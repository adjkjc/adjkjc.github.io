Follow Button Creation

Documentation → Follow Button Creation

The follow button is a small button that you can add to any site, that will allow WordPress.com users to easily follow the specified WordPress.com or Jetpack enabled blog.

Note: In order to generate the code for your Jetpack enabled blog, you must have the JSON API module activated on it.

Use the options below to generate the code for your WordPress.com Follow Button. You can use the generated code snippet to embed the Follow button anywhere you’d like to on the web.
