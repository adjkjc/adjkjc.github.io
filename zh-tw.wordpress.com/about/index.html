關於我們

Automattic 團隊，攝於加利福尼亞州聖克魯斯市

WordPress.com 的使命就是讓網站發佈逐漸大眾化。

開放原始碼的 WordPress 是最受歡迎的線上發佈平台。目前，在所有網路內容之中，使用 WordPress 發佈的網站所佔的比率已超過 20%。為了與更多人分享 WordPress 體驗，我們在 2005 年架設了 WordPress.com。

它也就是這套開放原始碼軟體的託管版本。即使你沒有任何與網路技術相關的專業知識，也能夠在這裡開始輕鬆架設網誌或網站。

Automattic 團隊的所有成員皆致力於持續改善 WordPress.com。我們幾乎每天都會進行更新，並積極開發其他服務，像是 Gravatar 和 Simplenote，以改善你的網路體驗。我們的目標是為你提供各種你喜愛的產品、功能和主題，因此請將你寶貴的意見回饋給我們。

WordPress.com 的服務幾乎全部免費，而目前免費提供的服務項目在未來也會維持免費。為了持續向你提供免費的網站服務，我們推出各種升級項目，其中包括自訂設計和自訂網域等服務，並提供反垃圾信件軟體 Akismet等產品，以及與主要媒體管道建立 VIP 託管合作關係。

無論你是部落客或網站擁有者，我們都能為你提供網路上的各種資源，讓你能夠輕鬆上手。如果你已經是 WordPress 使用者，感謝你選擇我們，很高興能夠為你服務。如果你打算提升你的網路影響力，但尚未決定該從哪裡下手的話，請試用我們的服務。我們很樂意成為你網路上的第二個家。
