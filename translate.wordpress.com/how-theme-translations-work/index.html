How Theme Translations Work

Anyone can contribute to WordPress.com theme translation. Visit the themes list page and start translating.

Once the number of “Current” translations reach at least 85% for a theme, we automatically deploy it once a day.

If it’s taking too long for your project validator to check the translations you suggested, please try contacting a validator. You can also let us know via the translation forums.

Meet the Translation Community

Stay in touch! Visit our community P2 at wpcomtranslation.wordpress.com for updates and discussions for active translators.
