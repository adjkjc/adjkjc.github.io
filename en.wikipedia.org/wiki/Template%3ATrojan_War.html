Template documentation[view] [edit] [history] [purge]

This template is used to provide a convenient means of navigating around articles on the Trojan War.

Contents

  • 1 Usage
  • 2 Parameters
  • 3 Example
  • 4 References

Usage[edit]

{{Trojan War}}

{{Trojan War|width=dddpx}}  [where ddd is a decimal number]

{{Trojan War|era=BCE}}

{{Trojan War|width=dddpx|era=BCE}} [where ddd is a decimal number]

Parameters[edit]

  • Parameters can be safely left blank.
  • width – the width of the navigation box. The default is 190px.
  • era – abbreviation to be used after dates in place of "BC", which is the default. On pages which use the CE/BCE convention for dates the value of this parameter should be set to BCE.
